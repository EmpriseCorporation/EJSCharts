EJSC.Chart.show_messages

See Also

 Top  Previous  Next

 

Definition

 

boolean show_messages = true

 

Description

 

Defines if progress messages should be displayed above the chart or not.

 

Example

 

>> Turn off progress messages.

 

var chart = new EJSC.Chart(

              "chart",

              {show_messages: false}

       );
