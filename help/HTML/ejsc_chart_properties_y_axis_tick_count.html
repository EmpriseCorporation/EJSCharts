EJSC.Chart.y_axis_tick_count

See Also

 

Definition

 

integer y_axis_tick_count = undefined

 

Description

 

Defines the number of ticks to be displayed on the Y-Axis.

 

Example

 

>> Show only 3 ticks on the Y-Axis at all times.

 

var chart = new EJSC.Chart(

              "chart",

              {y_axis_tick_count: 3}

       );
