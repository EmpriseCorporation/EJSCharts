EJSC.Chart.y_axis_max_tick_interval        

See Also

 

Definition

 

integer y_axis_max_tick_interval = undefined

 

Description

 

Defines the maximum interval between major tick marks / labels.  This will override the auto generation of ticks to cap the interval to the value when defined.

 

Example

 

>> Make the maximum interval between ticks 2, regardless of the data range

 

var chart =  new EJSC.Chart(

              "chart",

              { y_axis_max_tick_interval: 2 }

       );
